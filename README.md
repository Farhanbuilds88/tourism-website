# 🏔️ Tourism Website

## 📌 Project Overview
This is a responsive **Tourism Website** built using HTML, CSS, and JavaScript.  
It showcases beautiful tourist destinations with a modern and user-friendly interface.

The purpose of this project is to practice front-end development and build a real-world styled website.

---

## 🌍 Features
- Fully responsive design (mobile + desktop)
- Attractive landing page
- Tourist destination sections
- Image galleries
- Navigation bar
- Smooth scrolling
- Interactive UI using JavaScript
- Clean and modern layout

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
